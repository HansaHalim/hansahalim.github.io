Playing chess and want to know if you are losing? I made a checkmate checker where you can have 1, 2 or as many queens as the chess board allows. Usually when we play chess, it is hard to determine whether you have any possible moves left as a single king, which is why this program comes in helpful. This project is created as part of a CS136 assignment.

Traversing in C - Maze Solver

I wrote a maze solver that can solve a maze using right-hand rule wall follower algortithm. The image on the right is a maze question and the solution is given on it. The image on the left is a machine readable code for the maze. The solver then uses the wall follower algorithm to find the solution to this maze. The solution is then shown where U - Up, D - Down, R - Right, L - Left from our point of view. By following the solution set, it will lead you from the start line to the finish line. This project was created as part of a CS136 assignment.
