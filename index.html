Real Time Mobile Queuing System - Java

This project is an MLH Hackathon Winner for Best Internet of Things Category in StarterHacks 2019. We made a virtual queue system where people can line up for a service without having to be physically present.

The app was coded in Java and XML using Android Studio. I integrated the app with Google's Firebase Real Time Database to store vital information about the queue. In addition I also implemented the QR code mechanism between the business and the client using the ZXing library. I was the Lead Developer for this project with 3 other people.

This project is created as part of a CS136 assignment.

Traversing in C - Maze Solver

I wrote a maze solver that can solve a maze using right-hand rule wall follower algortithm. The image on the right is a maze question and the solution is given on it. The image on the left is a machine readable code for the maze. The solver then uses the wall follower algorithm to find the solution to this maze. The solution is then shown where U - Up, D - Down, R - Right, L - Left from our point of view. By following the solution set, it will lead you from the start line to the finish line. This project was created as part of a CS136 assignment.

Conway's Game of Life - C++
